# Zoolini Crush

Ein kleines Match-3-Spiel (Candy-Crush-Style) mit Hunden und Katzen, inspiriert von zoolini.de.

**Spielen:** https://nichtagentur.github.io/zoolini-crush/

## Spielregeln

- 8x8 Raster mit sechs Tierarten (drei Hunde, drei Katzen)
- Zwei benachbarte Tiere tauschen, drei oder mehr gleiche in einer Reihe loesen sich auf
- Kaskaden zaehlen extra Punkte
- 20 Zuege Zeit, Ziel: 600 Punkte
- Brett wird automatisch neu gemischt, wenn kein Zug mehr moeglich ist

Single-File. Reines HTML + Canvas + JavaScript, keine Dependencies.

Gebaut beim niceshops AI Computer Use Workshop (Saaz, 2026-05-27) mit Claude Code.
